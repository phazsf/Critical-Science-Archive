# Critical Science Archive

This repository hosts critical commentaries, figure reanalyses, and methodologically rigorous critiques of published papers. Our goal is to provide an open, reproducible platform for scrutinizing experimental designs, statistical rigor, and computational validity in high-impact research.

## Contents

- **/Critiques** - Full commentary PDFs and detailed analyses.
- **/Figures** - Images and figures from critiques.
- **/Scripts** - Python or R scripts used for figure analysis.
- **/Metadata** - Citation information and readme files.

## Contributing

We welcome contributions! If you find an article that needs re-evaluation, feel free to fork the repository, make your edits, and submit a pull request.
